布里塔尼亚的谎言对决 - 基于《Code Geass》世界观的卡牌对战游戏

**[点击这里开始游戏](https://twmissingu.github.io/liars-game/)**

## 游戏简介

《Code Geass: Liar's Game》是一款结合了策略、欺骗和Geass能力的卡牌对战游戏。玩家将扮演《Code Geass》中的经典角色，与3个AI对手进行激烈的谎言对决。

## 游戏特色

### 🎭 4个可玩角色
每个角色都有独特的技能：

- **鲁鲁修** - 绝对命令：可以强制改变骗子牌
- **C.C.** - 不老不死：50%概率免疫Geass效果
- **朱雀** - 生存本能：HP≤1时Geass命中概率减半
- **卡莲** - 红莲突击：每回合可出1-4张牌

### 🎯 1v3人机对战
- 同时对抗3个AI对手
- 每个AI都有不同的角色和策略
- 最后存活的玩家获胜

### 🃏 标准扑克牌系统
- 使用52张标准扑克牌
- 每回合随机指定"骗子牌"
- 可以出真牌或撒谎

### ⚡ Geass判定系统
- 质疑成功触发Geass判定
- 基础1/3概率命中
- 角色技能影响概率
- 8种搞笑动作效果

### 🎵 完整音效系统
- BGM背景音乐
- 出牌音效
- Geass激活音效
- 角色技能音效
- 搞笑动作音效

## 游戏规则

1. **初始发牌**：每人5张牌
2. **骗子牌**：每回合随机指定一张骗子牌（A-K）
3. **出牌**：轮流出牌，可以声称是骗子牌（可能撒谎）
4. **质疑**：其他玩家可以质疑出牌
5. **Geass判定**：质疑成功触发Geass判定，有概率造成1点伤害
6. **胜利条件**：淘汰所有对手，成为最后幸存者

## 角色技能详解

### 鲁鲁修 - 绝对命令
- **效果**：可以强制改变骗子牌为任意点数
- **使用时机**：自己回合开始时
- **策略**：可以配合手牌改变局势

### C.C. - 不老不死
- **效果**：受到Geass时有50%概率免疫
- **被动技能**：自动触发
- **策略**：可以更大胆地撒谎

### 朱雀 - 生存本能
- **效果**：HP≤1时，Geass命中概率减半（从1/3降至1/6）
- **被动技能**：危急时刻自动触发
- **策略**：残血时有更强的生存能力

### 卡莲 - 红莲突击
- **效果**：每回合可出1-4张牌
- **其他角色**：每回合只能出1张牌
- **策略**：可以快速出完手牌或制造压力

## 技术栈

- React 18 + TypeScript
- Vite 构建工具
- Howler.js 音效管理
- CSS3 动画效果

## 本地开发

```bash
# 克隆仓库
git clone https://github.com/twmissingu/liars-game.git
cd liars-game

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 部署到GitHub Pages
npm run deploy
```
