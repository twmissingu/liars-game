**版本**: v1.0.0  
**日期**: 2026-03-14  
**适用游戏版本**: v2.0.0+

## 1. 游戏简介

《Code Geass: Liar's Game》是一款基于《Code Geass》世界观的卡牌策略游戏。你将扮演动画中的经典角色，与3名AI对手进行心理博弈，运用策略和谎言击败对手！

### 1.1 游戏特色
- 🎭 **Code Geass正版角色**: 鲁鲁修、C.C.、朱雀、卡莲
- 🃏 **骗子牌玩法**: 出真牌还是撒谎？由你决定
- ⚡ **Geass系统**: 独特的技能机制，改变战局
- 🤖 **智能AI**: 3个不同性格的AI对手
- 🎨 **精美画面**: 还原动画风格的视觉体验

---

## 2. 快速入门

### 2.1 开始游戏

1. **打开游戏**: 在浏览器中访问游戏地址
2. **点击开始**: 在主菜单点击"开始游戏"按钮
3. **选择角色**: 从4个角色中选择你喜欢的角色
4. **确认选择**: 点击"确认"进入游戏

### 2.2 游戏目标

**击败所有3个AI对手，成为最后的胜者！**

每个玩家初始有3点HP，HP归零即被淘汰。通过出牌和质疑让对手接受Geass判定，削减他们的HP。

## 3. 游戏规则详解

### 3.1 卡牌系统

#### 3.1.1 卡牌构成
- **牌组**: 使用Q、K、A三种点数的扑克牌
- **花色**: ♠黑桃、♥红桃、♣梅花、♦方块
- **特殊牌**: 小丑牌(JOKER)可以当作任意点数使用

#### 3.1.2 手牌
- 游戏开始时每人获得5张牌
- 手牌只有你自己能看到
- 出完手牌后需要重新发牌

### 3.2 骗子牌机制

#### 3.2.1 什么是骗子牌？
每回合开始时，系统会随机指定一种点数（Q、K或A）为**骗子牌**。

#### 3.2.2 出牌规则
- 你必须声称自己出的牌是骗子牌
- 你可以选择：
  - **出真牌**: 出的确实是骗子牌（诚实）
  - **撒谎**: 出的不是骗子牌，但声称是（欺骗）

**示例**:
- 骗子牌是 **K**
- 你手中有 **♠K** 和 **♥Q**
- 你可以出 **♠K**（真牌）或 **♥Q**（撒谎）

### 3.3 回合流程

```

│                      一个完整的回合                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │

│                                                             │
│  2️⃣ 出牌阶段（当前玩家）                                     │
│     ├── 选择1-3张手牌（卡莲可选1-4张）                       │
│     ├── 系统自动声称是骗子牌                                │
│     └── 点击"出牌"按钮                                      │

│     ├── 每个玩家依次决定是否质疑                            │

│     └── 未命中 → 无事发生                                   │

│     └── 轮到下一位玩家                                      │
```

### 3.4 质疑系统

#### 3.4.1 什么时候质疑？
当你**怀疑**对方出的牌不是骗子牌时，可以选择质疑！

#### 3.4.2 质疑结果

| 情况 | 结果 | 后果 |
|------|------|------|
| 质疑成功 | 对方在撒谎 | **对方**接受Geass判定 |
| 质疑失败 | 对方诚实 | **你**接受Geass判定 |
| 无人质疑 | - | 回合继续，牌进入弃牌堆 |

**质疑策略提示**:
- 如果你手中有骗子牌，对方却声称出了骗子牌，很可能在撒谎！
- 残血时（HP=1）要谨慎质疑，避免自己受Geass
- 观察对手的出牌习惯，找出规律

### 3.5 Geass判定系统

#### 3.5.1 基础机制
- **命中率**: 33.33%（1/3概率）
- **伤害**: 命中造成1点HP伤害
- **最大HP**: 每人3点HP

#### 3.5.2 搞笑动作
Geass命中时，目标会执行随机搞笑动作：

| 动作 | 描述 |
|------|------|
| 😵 | 突然跳起了奇怪的舞蹈 |
| 🙈 | 开始模仿猴子叫 |
| 🤪 | 不停地说"披萨" |
| 😂 | 无法控制地大笑30秒 |
| 🐔 | 学鸡打鸣 |
| 🎭 | 开始背诵中二台词 |
| 🍕 | 声称自己是披萨的化身 |
| 🦋 | 追逐不存在的蝴蝶 |

## 4. 角色技能详解

### 4.1 鲁鲁修 - 绝对命令

#### 技能效果
强制改变当前骗子牌为任意指定点数（Q/K/A）。

#### 使用时机
- 你的回合开始时，出牌前
- 每局限用1次

#### 使用技巧
- **配合手牌**: 如果你有很多Q，就把骗子牌改成Q
- **迷惑对手**: 改变骗子牌让对手的判断失误
- **关键时刻**: 在残血时使用，增加生存机会

#### 技能台词
> "我以鲁鲁修·vi·布里塔尼亚之名下令..."

---

### 4.2 C.C. - Code之力

#### 技能效果
首次HP归零时，有50%概率复活并恢复1点HP。

#### 特点
- **被动技能**: 自动触发，无需操作
- **每局限1次**: 复活后技能失效
- **高容错**: 给新手更多容错空间

#### 使用技巧
- **敢于冒险**: 有复活保底，可以更积极质疑
- **心理战术**: 对手知道你可能复活，不敢轻易质疑你

#### 技能台词
> "我是不老不死的魔女..."

---

### 4.3 朱雀 - 枢木剑术

#### 技能效果
- **闪避**: 15%概率完全闪避Geass
- **反击**: 闪避成功时，25%概率反击，使攻击者受Geass

#### 特点
- **防御型**: 擅长防守反击
- **残血强势**: HP越低越要小心朱雀

#### 使用技巧
- **引诱质疑**: 故意出可疑的牌，引诱对手质疑后反击
- **持久战**: 利用高闪避率消耗对手

#### 技能台词
> "我要从内部改变世界！"

---

### 4.4 卡莲 - 红莲二式

#### 技能效果
1. **多牌出牌**: 每回合可出1-4张牌（其他角色最多3张）
2. **Geass强化**: 出2张以上牌且质疑失败时，Geass命中率提升
   - 出2张: 40%命中率
   - 出3张: 60%命中率
   - 出4张: 80%命中率

#### 特点
- **高风险高回报**: 出牌越多，被质疑成功的损失越大
- **进攻型**: 适合积极进攻的玩家

#### 使用技巧
- **快速清牌**: 利用多牌优势快速出完手牌
- **威慑战术**: 出多张牌时，对手不敢轻易质疑
- **精准计算**: 出2张性价比最高，命中率提升且风险可控

#### 技能台词
> "黑色骑士团，突入！"

## 6. 操作指南

### 6.1 基本操作

#### 6.1.1 选择牌
- **鼠标**: 点击手牌进行选择/取消
- **触控**: 点击手牌进行选择/取消
- **键盘**: 数字键1-5选择对应位置的牌

#### 6.1.2 出牌
1. 点击选择要出的牌（1-3张，卡莲可1-4张）
2. 点击"出牌"按钮
3. 等待其他玩家质疑

#### 6.1.3 质疑
- 当其他玩家出牌后，你可以选择：
  - **质疑**: 点击"质疑"按钮，怀疑对方撒谎
  - **不质疑**: 点击"跳过"按钮，相信对方

#### 6.1.4 使用技能
- **鲁鲁修**: 回合开始时点击技能按钮，选择新的骗子牌
- **其他角色**: 被动技能自动触发，无需操作

### 6.2 快捷键

| 快捷键 | 功能 |
|--------|------|
| 1-5 | 选择第1-5张手牌 |
| Space | 出牌/确认 |
| C | 质疑 |
| P | 跳过 |
| S | 使用技能（鲁鲁修） |
| M | 静音切换 |
| Esc | 返回/取消 |

### 6.3 设置选项

#### 6.3.1 音量设置
- **BGM音量**: 调整背景音乐音量
- **音效音量**: 调整游戏音效音量
- **静音**: 一键静音所有声音

#### 6.3.2 游戏设置
- **AI难度**: 选择Easy/Normal/Hard
- **动画速度**: 调整动画播放速度
- **显示日志**: 开启/关闭游戏日志

## 7. 游戏策略

### 7.1 新手建议

1. **先熟悉规则**: 前几局专注于理解游戏流程
2. **选择C.C.**: 新手推荐C.C.，有复活技能容错
3. **谨慎质疑**: 不确定时不要轻易质疑
4. **观察对手**: 注意AI的出牌习惯

### 7.2 进阶技巧

#### 7.2.1 出牌策略
- **有真牌出真牌**: 降低被质疑风险
- **无真牌时**: 考虑出单张撒谎，风险较小
- **残血时**: 尽量出真牌，避免被质疑成功

#### 7.2.2 质疑策略
- **手中有骗子牌**: 对方声称出骗子牌时，大概率在撒谎
- **观察出牌数量**: 一次出多张牌时，更可能在撒谎
- **记忆对手手牌**: 通过游戏日志推测对手剩余牌

#### 7.2.3 角色配合
- **鲁鲁修**: 在关键时刻改变骗子牌，配合手牌清场
- **C.C.**: 利用复活心理，引诱对手质疑后反杀
- **朱雀**: 残血时故意出可疑牌，引诱质疑后反击
- **卡莲**: 快速出完手牌，减少被攻击机会

### 7.3 常见错误

| 错误 | 说明 | 改进建议 |
|------|------|----------|
| 过度质疑 | 频繁质疑导致自己受Geass | 提高质疑准确率 |
| 总是诚实 | 从不出撒谎，失去策略性 | 适当撒谎迷惑对手 |
| 忽视技能 | 忘记使用角色技能 | 熟悉技能时机 |
| 盲目出牌 | 不考虑对手手牌 | 观察和分析对手 |

## 8. 常见问题 (FAQ)

### Q1: 游戏无法加载怎么办？
**A**: 
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试使用Chrome或Firefox浏览器
4. 确保浏览器已启用JavaScript

### Q2: 为什么我无法质疑？
**A**: 
- 只有其他玩家出牌后你才能质疑
- 如果是你的回合，你只能出牌
- 检查是否已经进入质疑阶段

### Q3: 角色技能如何触发？
**A**: 
- **鲁鲁修**: 回合开始时点击技能按钮
- **C.C./朱雀/卡莲**: 被动技能自动触发

### Q4: 游戏支持多人对战吗？
**A**: 
当前版本仅支持1v3人机对战。联网对战功能正在开发中。

### Q5: 如何调整游戏难度？
**A**: 
在主菜单的"游戏设置"中，可以调整AI难度（Easy/Normal/Hard）。

### Q6: 游戏进度会保存吗？
**A**: 
每局游戏独立进行，不支持存档。一局通常5-10分钟。

### Q7: 为什么我的牌出不去？
**A**: 
- 检查是否是你的回合
- 检查是否已选择牌
- 检查出牌数量是否超过限制

### Q8: 如何静音？
**A**: 
点击顶部栏的音量图标，或在设置中调整音量。

## 9. 故障排除

### 9.1 性能问题

| 问题 | 解决方案 |
|------|----------|
| 游戏卡顿 | 关闭其他标签页，降低动画质量 |
| 加载缓慢 | 检查网络，清除缓存 |
| 音效延迟 | 刷新页面，检查音频驱动 |

### 9.2 显示问题

| 问题 | 解决方案 |

| 界面显示不全 | 调整浏览器缩放比例（Ctrl+滚轮）|
| 图片不显示 | 刷新页面，检查网络 |
| 文字模糊 | 调整浏览器缩放至100% |
