## 发现的问题与修复

### 问题1: 质疑阶段空值检查
**问题描述**: 当所有人都选择不质疑时，`passToNextPlayer`会清空`playedCards`，导致后续质疑调用出错。

**修复方案**: 
- 在测试代码中添加空值检查
- 在调用`playerChallengeDecision`前验证`playedCards`存在

```typescript
if (!state.turnState.playedCards) {
  console.log('没有出牌记录，跳过质疑阶段');
  continue;
}
```

### 问题2: 性能测试循环
**问题描述**: 性能测试中的快速循环可能导致状态不一致。

**修复方案**: 添加状态检查，确保在有效状态下才执行操作。
