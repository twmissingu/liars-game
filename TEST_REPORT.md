- **描述**: 卡莲技能加成计算时，0.2 * 3 = 0.6000000000000001
- **修复**: 使用 `toBeCloseTo` 替代 `toBe` 进行浮点数比较
- **状态**: ✅ 已修复

### 问题2：测试环境回合状态
- **描述**: 部分测试需要确保是玩家回合才能出牌
- **修复**: 添加回合状态检查和设置
- **状态**: ✅ 已修复
