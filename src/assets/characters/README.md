## 角色设计说明

### 鲁鲁修 (Lelouch)
- **主色调**: 黑紫色 (#4c1d95)
- **特征**: 眼罩（遮住Geass之眼）、黑色披风
- **姿势**: 思考姿势 - 单手托腮
- **动画**: 呼吸、思考、出牌、胜利、失败

### C.C.
- **主色调**: 绿色 (#15803d)
- **特征**: 披萨、白色拘束服、绿色长发
- **姿势**: 慵懒姿势 - 手持披萨，侧躺
- **动画**: 呼吸、慵懒、出牌、胜利、失败

### 朱雀 (Suzaku)
- **主色调**: 白色 (#f8fafc)
- **特征**: 骑士制服、兰斯洛特徽章
- **姿势**: 热血姿势 - 握拳，充满干劲
- **动画**: 呼吸、热血、出牌、胜利、失败

### 卡莲 (Kallen)
- **主色调**: 红色 (#dc2626)
- **特征**: 红色头巾、驾驶服、红莲徽章
- **姿势**: 傲娇姿势 - 双臂交叉，侧头
- **动画**: 呼吸、傲娇、出牌、胜利、失败

## 使用说明

角色组件已使用内联SVG实现，无需外部素材文件。
组件位置: `src/components/characters/`
